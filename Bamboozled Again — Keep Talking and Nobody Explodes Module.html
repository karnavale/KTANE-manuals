Keep Talking and Nobody Explodes Mod Снова одурачен

О модуле «Снова одурачен»

"Wait, was that the letter Echo or the word E?", you say,
as you are baffled, befuddled, and bemused.
This module will beguile, and buffalo, and bewilder,
to make sure it is never defused.

Данный модуль содержит 6 цветных кнопок с текстом на каждой из них и экран, который показывает сообщение, поделённое на 8 частей.
Каждая часть сообщения зашифрована с помощью трёх ключевых значений A, B и C следующим образом:

  1. Каждый символ, включая пробелы, заменяется символом, находящимся на A позиций левее.
    (0 <= A < Количество символов в тексте)
  2. Пара символов добавляется в конец текста.
  3. Каждый символ заменяется символом, находящимся на B букв/символов правее. (0 < B <= 26)
  4. Текст записывается одним из шести наборов глифов. Выбранный набор даст значение для C.

Каждый незашифрованный текст, не включая 2-ой и 4-ый, содержит значение, изменяемое оператором, соответствующим цвету текста.

Все эти значения, вместе с A, B и C, дадут конечное значение каждого из текстов.

У каждой кнопки имеется начальное значение, заданное её цветом и текстом на ней.

Все эти значения вместе с финальными значениями текстов на экране дадут конечное значение каждой из кнопок.

Верные кнопки, которые необходимо нажать, определяются их конечными значениями.

Используйте текст на кнопках, их цвет, и символы, добавленные к каждому из текстов на экране на 2-ом этапе шифрования, чтобы найти правильные моменты времени, когда необходимо нажать на кнопки.

Если вы нажмёте на кнопку, загорится светодиод. Как только загорятся все четыре светодиода, будут отправлены ответы.

    Светодиоды поменяют свой цвет в зависимости от отправленных ответов:
  • Зелёный - Верная кнопка была нажата в правильное время.
  • Жёлтый - Верная кнопка была нажата в неправильное время.
  • Красный - Была нажата неправильная кнопка.

Если все 4 светодиода загорятся зелёным, модуль будет деактивирован. Если ни один из светодиодов не загорелся красным, модуль обнулится, однако ошибка не будет зафиксирована.
В противном случае модуль обнулится, и будет зафиксирована ошибка.

Дополнительная информация по модулю:

Светодиоды можно нажать в любое время, чтобы повлиять на экран:

  • Первый слева - Переходит к предыдущему тексту в режиме паузы.
  • Второй слева - Возобновляет автоматическую текстовую прокрутку.
  • Второй справа - Приостанавливает текстовую прокрутку.
  • Первый справа - Переходит к следующему тексту в режиме паузы.

Подчасть 1.4: Конечные значения текстов

Конечное значение T для каждого из 6 текстов вычисляется по формуле:

T = S + 5*A + 2(B + C)

Подчасть 1.5: Начальные значения кнопок

Следуйте инструкциям ниже, чтобы вычислить начальное значение I каждой из шести кнопок:

    • Если кнопка чёрного цвета, изначально I равно 30.
    • Иначе, если кнопка белого или серого цвета, изначально I равно 20.
    • Иначе изначально I равно 0.
    • Если на кнопке написан её цвет, прибавьте 70.
    • Иначе, если на кнопке написан дополнительный цвет, прибавьте 35.
    • Иначе, если на кнопке написан любой цвет или слово “COLOUR”, прибавьте 5.
  3. Прибавьте 60 за каждый незашифрованный текст на экране, который совпадает с написанным на кнопке.
  4. Прибавьте 15 за каждый текст на экране, цвет которого совпадает с цветом на кнопке.
  5. Если кнопка не серого цвета, прибавьте 10 за каждый текст на экране, чей цвет является дополнительным к цвету кнопки.

Subsection 2.1.2: Y value computation

  • If there are no lit LEDs, the Y value is the current X value.
  • If there is one lit LED and, once deciphered,
    • display text 2 is THEN, Y is the current X value plus the previous X value.
    • display text 2 is NEXT, Y is the current X value minus the previous X value.
  • If there are two lit LEDs and, once deciphered,
    • display text 4 is THEN, Y is the current X value plus the previous Y value.
    • display text 4 is NEXT, Y is the current X value minus the previous Y value.

Subsection 2.1.3: Final computation

The time to push the button is given by the Y value and the symbols appended to the text at step 2 of the the encryption.

  • If there are no lit LEDs, use the symbols appended to display text 8.
  • If there is one lit LED, use the symbols appended to display text 7.
  • If there are two lit LEDs, use the symbols appended to display text 6.
