О Микроконтроллере

Это называется «микро»-контроллер, однако эта штука довольно огромна внутри. Возможно потому что может устроить нехилый взрыв...

  1. Используя название, напечатанное на микроконтроллере, и его размер, определите конфигурацию контакта по диаграммам ниже.
  2. Белая отметка на контроллере обозначает, где расположен контакт с номером 1. Остальные контакты расположены по возрастанию на стороне с номером 1, а затем идут в обратном порядке на другой стороне.
  3. По таблице ниже определите верный цветовой код для каждого подсоединённого элемента.
  4. Для каждого из контактов выберите верный элемент, нажимая кнопки “UP” и “DOWN”, и отправьте ответ с помощью кнопки “OK” (следующий контакт будет выбран автоматически).
